# Performance Expert Skill Installation

## Installation

### Via Symlink (Recommended)

```bash
ln -s /path/to/skills/.cursor/skills/performance-expert ~/.cursor/skills/performance-expert
```

### Via Copy

```bash
cp -r /path/to/skills/.cursor/skills/performance-expert ~/.cursor/skills/performance-expert
```

## Verification

```bash
ls -la ~/.cursor/skills/performance-expert/SKILL.md
```

## Usage

This skill activates automatically when you're working on:
- Performance optimization
- Bundle size analysis
- Database query optimization
- Caching strategies
- Core Web Vitals
- API response times

No manual activation needed - Claude will load this skill based on context.
