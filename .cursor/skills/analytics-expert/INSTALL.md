# Installation: Analytics Expert

## Prerequisites
- Cursor installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.cursor/skills/analytics-expert ~/.cursor/skills/analytics-expert
```

## Install via Copy
```bash
cp -r /path/to/skills/.cursor/skills/analytics-expert ~/.cursor/skills/analytics-expert
```

## Verify Installation
```bash
ls -la ~/.cursor/skills/analytics-expert
cat ~/.cursor/skills/analytics-expert/SKILL.md
```

## Usage
The skill activates automatically when you ask analytics questions in Cursor.

None
