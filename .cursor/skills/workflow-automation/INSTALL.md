# Installation: Workflow Automation

## Prerequisites

- Cursor installed
- Clone this repository

## Install via Symlink

```bash
ln -s /path/to/skills/.cursor/skills/workflow-automation ~/.cursor/skills/workflow-automation
```

## Install via Copy

```bash
cp -r /path/to/skills/.cursor/skills/workflow-automation ~/.cursor/skills/workflow-automation
```

## Verify Installation

```bash
ls -la ~/.cursor/skills/workflow-automation
cat ~/.cursor/skills/workflow-automation/SKILL.md
```

## Usage

The skill activates automatically when you need help designing workflows, creating process documentation, implementing automation rules, or optimizing content pipelines.
