# Installation: Roadmap Analyzer

## Prerequisites
- Claude Code installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.claude/skills/roadmap-analyzer ~/.claude/skills/roadmap-analyzer
```

## Install via Copy
```bash
cp -r /path/to/skills/.claude/skills/roadmap-analyzer ~/.claude/skills/roadmap-analyzer
```

## Verify Installation
```bash
ls -la ~/.claude/skills/roadmap-analyzer
cat ~/.claude/skills/roadmap-analyzer/SKILL.md
```

## Usage
The skill activates automatically when you need to analyze features against ICP needs, identify gaps, and recommend roadmap priorities.
