# Installation: QA Reviewer

## Prerequisites
- Claude Code installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.claude/skills/qa-reviewer ~/.claude/skills/qa-reviewer
```

## Install via Copy
```bash
cp -r /path/to/skills/.claude/skills/qa-reviewer ~/.claude/skills/qa-reviewer
```

## Verify Installation
```bash
ls -la ~/.claude/skills/qa-reviewer
cat ~/.claude/skills/qa-reviewer/SKILL.md
```

## Usage
The skill activates automatically when you need to systematically review AI agent work for quality, accuracy, and completeness.
