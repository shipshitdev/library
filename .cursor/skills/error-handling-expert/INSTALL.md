# Error Handling Expert Skill Installation

## Installation

### Via Symlink (Recommended)

```bash
ln -s /path/to/skills/.cursor/skills/error-handling-expert ~/.cursor/skills/error-handling-expert
```

### Via Copy

```bash
cp -r /path/to/skills/.cursor/skills/error-handling-expert ~/.cursor/skills/error-handling-expert
```

## Verification

```bash
ls -la ~/.cursor/skills/error-handling-expert/SKILL.md
```

## Usage

This skill activates automatically when you're working on:
- Error handling implementation
- Exception filters
- Error responses
- Error logging
- Error recovery

No manual activation needed - Claude will load this skill based on context.
