# Installation: Planning Assistant

## Prerequisites

- Cursor installed
- Clone this repository

## Install via Symlink

```bash
ln -s /path/to/skills/.cursor/skills/planning-assistant ~/.cursor/skills/planning-assistant
```

## Install via Copy

```bash
cp -r /path/to/skills/.cursor/skills/planning-assistant ~/.cursor/skills/planning-assistant
```

## Verify Installation

```bash
ls -la ~/.cursor/skills/planning-assistant
cat ~/.cursor/skills/planning-assistant/SKILL.md
```

## Usage

The skill activates automatically when you need help with content planning, calendar management, or research organization.
