# Installation: Design Consistency Auditor

## Prerequisites
- Claude Code installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.claude/skills/design-consistency-auditor ~/.claude/skills/design-consistency-auditor
```

## Install via Copy
```bash
cp -r /path/to/skills/.claude/skills/design-consistency-auditor ~/.claude/skills/design-consistency-auditor
```

## Verify Installation
```bash
ls -la ~/.claude/skills/design-consistency-auditor
cat ~/.claude/skills/design-consistency-auditor/SKILL.md
```

## Usage
The skill activates automatically when you need to audit design system consistency, UX/UI patterns, and design best practices.
