# Installation: Session Documenter

## Prerequisites
- Claude Code installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.claude/skills/session-documenter ~/.claude/skills/session-documenter
```

## Install via Copy
```bash
cp -r /path/to/skills/.claude/skills/session-documenter ~/.claude/skills/session-documenter
```

## Verify Installation
```bash
ls -la ~/.claude/skills/session-documenter
cat ~/.claude/skills/session-documenter/SKILL.md
```

## Usage
The skill activates automatically to document session work, decisions, and file changes for session continuity.
