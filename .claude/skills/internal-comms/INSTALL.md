# Installation: Internal Comms

## Prerequisites
- Claude Code installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.claude/skills/internal-comms ~/.claude/skills/internal-comms
```

## Install via Copy
```bash
cp -r /path/to/skills/.claude/skills/internal-comms ~/.claude/skills/internal-comms
```

## Verify Installation
```bash
ls -la ~/.claude/skills/internal-comms
cat ~/.claude/skills/internal-comms/SKILL.md
```

## Usage
The skill activates automatically when you need to write internal communications such as status reports, leadership updates, 3P updates, company newsletters, or FAQs.
