# Ship Shit Dev - Content Bundle

Content creation and documentation skills

## Installation

```bash
/plugin marketplace add shipshitdev/library
/plugin install shipshitdev-content@shipshitdev
```

## Included Skills

- `copywriter`
- `content-creator`
- `nextra-writer`
- `docs`
- `changelog-generator`
- `internal-comms`
- `youtube-video-analyst`
