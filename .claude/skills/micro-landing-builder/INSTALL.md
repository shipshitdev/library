# Installation: Micro Landing Builder

## Prerequisites
- Claude Code installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.claude/skills/micro-landing-builder ~/.claude/skills/micro-landing-builder
```

## Install via Copy
```bash
cp -r /path/to/skills/.claude/skills/micro-landing-builder ~/.claude/skills/micro-landing-builder
```

## Verify Installation
```bash
ls -la ~/.claude/skills/micro-landing-builder
cat ~/.claude/skills/micro-landing-builder/SKILL.md
```

## Usage
The skill activates automatically when you need to scaffold config-driven NextJS landing pages for startups.

## Dependencies
- Python 3 (for scaffold script)
- Published landing UI components package
