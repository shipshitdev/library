# Installation: Agent Folder Init

## Prerequisites
- Claude Code installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.claude/skills/agent-folder-init ~/.claude/skills/agent-folder-init
```

## Install via Copy
```bash
cp -r /path/to/skills/.claude/skills/agent-folder-init ~/.claude/skills/agent-folder-init
```

## Verify Installation
```bash
ls -la ~/.claude/skills/agent-folder-init
cat ~/.claude/skills/agent-folder-init/SKILL.md
```

## Usage
The skill activates automatically when you need to initialize a comprehensive `.agent/` folder structure for AI-first development workflows.

## Dependencies
- Python 3 (for scaffold script)
