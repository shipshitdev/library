# Ship Shit Dev - Planning Bundle

Strategy, planning, and analysis skills

## Installation

```bash
/plugin marketplace add shipshitdev/library
/plugin install shipshitdev-planning@shipshitdev
```

- `planning-assistant`
- `roadmap-analyzer`
- `strategy-expert`
- `analytics-expert`
- `cto-advisor`
- `business-operator`
- `task-prd-creator`
- `rules-capture`
