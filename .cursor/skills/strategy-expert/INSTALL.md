# Installation: Strategy Expert

## Prerequisites
- Cursor installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.cursor/skills/strategy-expert ~/.cursor/skills/strategy-expert
```

## Install via Copy
```bash
cp -r /path/to/skills/.cursor/skills/strategy-expert ~/.cursor/skills/strategy-expert
```

## Verify Installation
```bash
ls -la ~/.cursor/skills/strategy-expert
cat ~/.cursor/skills/strategy-expert/SKILL.md
```

## Usage
The skill activates automatically when you need help with content strategy, persona building, competitive analysis, or brand voice consistency.
