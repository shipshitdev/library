# Ship Shit Dev - Branding Bundle

Brand strategy and naming skills

## Installation

```bash
/plugin marketplace add shipshitdev/library
/plugin install shipshitdev-branding@shipshitdev
```

## Included Skills

- `brand-architect`
- `brand-name-generator`
- `search-domain-validator`
- `expert-architect`
- `expert-validator`
- `positioning-angles`
