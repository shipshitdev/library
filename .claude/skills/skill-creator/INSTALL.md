# Installation: Skill Creator

## Prerequisites
- Claude Code installed
- Clone this repository

## Install via Symlink
```bash
ln -s /path/to/skills/.claude/skills/skill-creator ~/.claude/skills/skill-creator
```

## Install via Copy
```bash
cp -r /path/to/skills/.claude/skills/skill-creator ~/.claude/skills/skill-creator
```

## Verify Installation
```bash
ls -la ~/.claude/skills/skill-creator
cat ~/.claude/skills/skill-creator/SKILL.md
```

## Usage
The skill activates automatically when you want to create a new skill or update an existing skill that extends Claude's capabilities.

## Dependencies
- Python 3 (for helper scripts)
