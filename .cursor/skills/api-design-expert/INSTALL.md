# API Design Expert Skill Installation

## Installation

### Via Symlink (Recommended)

```bash
ln -s /path/to/skills/.cursor/skills/api-design-expert ~/.cursor/skills/api-design-expert
```

### Via Copy

```bash
cp -r /path/to/skills/.cursor/skills/api-design-expert ~/.cursor/skills/api-design-expert
```

## Verification

```bash
ls -la ~/.cursor/skills/api-design-expert/SKILL.md
```

## Usage

This skill activates automatically when you're working on:
- API endpoint design
- RESTful API creation
- OpenAPI/Swagger documentation
- API versioning
- Error handling design

No manual activation needed - Claude will load this skill based on context.
