# Optimize Prompt - Improve AI Prompts

Optimize prompts for better AI model responses.

## When to Use

- Prompt giving poor results
- Need more consistent output
- Improving prompt templates
- Debugging AI behavior

## Optimization Steps

### Step 1: Analyze Current Prompt

Review the existing prompt:
- What is the goal?
- What format is expected?
- What context is provided?

### Step 2: Identify Issues

Common prompt problems:
- Too vague
- Missing context
- Unclear output format
- No examples
- Conflicting instructions

### Step 3: Apply Improvements

Prompt improvement techniques:
- Add clear role/persona
- Specify output format
- Include examples (few-shot)
- Break complex tasks into steps
- Add constraints and guidelines

### Step 4: Test Variations

Try different approaches:
- A/B test prompt versions
- Compare outputs
- Iterate on best performer

### Step 5: Document

Record the optimized prompt:
- What changed
- Why it improved
- When to use

## Prompt Best Practices

Structure:
1. Role/Context - Who is the AI?
2. Task - What should it do?
3. Format - How should output look?
4. Examples - Show expected output
5. Constraints - What to avoid

Clarity:
- Be specific, not vague
- Use simple language
- One instruction per line
- Avoid ambiguity

Examples:
- Include 2-3 examples
- Show edge cases
- Demonstrate format

## Common Fixes

Problem: Inconsistent format
Fix: Add explicit format specification with example

Problem: Missing information
Fix: Add required fields list

Problem: Wrong tone
Fix: Specify persona and communication style

Problem: Too verbose
Fix: Add word/length limits

Problem: Off-topic responses
Fix: Add focus constraints
