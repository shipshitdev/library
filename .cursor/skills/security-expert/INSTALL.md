# Security Expert Skill Installation

## Installation

### Via Symlink (Recommended)

```bash
ln -s /path/to/skills/.cursor/skills/security-expert ~/.cursor/skills/security-expert
```

### Via Copy

```bash
cp -r /path/to/skills/.cursor/skills/security-expert ~/.cursor/skills/security-expert
```

## Verification

```bash
ls -la ~/.cursor/skills/security-expert/SKILL.md
```

## Usage

This skill activates automatically when you're working on:
- Authentication/authorization
- Security configurations
- Input validation
- Data protection
- Security reviews

No manual activation needed - Claude will load this skill based on context.
