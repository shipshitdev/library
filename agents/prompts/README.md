# Prompts

Reusable prompts for AI coding assistants.

## Usage

Copy the prompt content and paste into your AI assistant, replacing any `@FILE` references with the actual file.

## Prompts

| Prompt | Description |
|--------|-------------|
| [prd-interview.md](./prd-interview.md) | Deep-dive interview for spec files. Asks non-obvious questions about technical implementation, UI/UX, concerns, and tradeoffs until the spec is complete. |
